.. note::

    **ScyllaDB Compatibility:** This backend is fully compatible with ScyllaDB.
    ScyllaDB is a drop-in replacement for Cassandra and works with the same
    driver and configuration. Use ``result_backend = 'scylladb://'`` or
    ``result_backend = 'cassandra://'`` - both work identically with ScyllaDB servers.
